Goal:
NP Coach helps Swedish students prepare for national exams.

However, the architecture must support multiple subjects and grades in the future.

TECH STACK:
- Frontend: React (clean, modern, minimal UI)
- Backend: Node.js + Express
- Database: JSON file or lightweight database (structured for scalability)
- Mobile responsive
- All UI text must be in Swedish

APP STRUCTURE REQUIREMENTS:

The system must be built using this hierarchy:

Subject
  -> Grade
     -> Test
        -> Questions

Even if we only implement:
- Subject: Matematik
- Grade: Åk 9

The structure must allow easy addition of:
- Svenska
- Engelska
- Other grades
